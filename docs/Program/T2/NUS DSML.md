???+ abstract "项目优劣势"

    Pros:
    
    * 学校层次、环境、花费，移民、人身安全类似cs学院的NUS Mcomp项目
    * 本项目有提前批，如果顺利的话8-9月就能拿到offer
    * 作为 NUS Mcomp的替补，CE可以拿来保底
    
    Cons:
    
    * 留位费比较高
    * 时长只有1年，相比Mcomp来说短一点
    * 新加坡的物价又涨了。。。
